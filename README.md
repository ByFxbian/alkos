# alkosbarber.at

Production website for **Alkos Barber** (Vienna).
Built with a focus on **speed, SEO, and clean UX**.

## Tech
- Next.js
- React + TypeScript
- Tailwind CSS

## Features
- Responsive layout (mobile-first)
- SEO-friendly pages / metadata
- Fast loading & optimized assets
- Simple content structure for quick updates

## Local Setup
```bash
pnpm install
pnpm dev
```

## Notes
This repo contains the public website. Some internal client tooling and assets are not included.
